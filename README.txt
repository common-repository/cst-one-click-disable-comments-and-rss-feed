One click to disable the entire comment system and RSS feeds. Allows you to hide all references to comments on the site.

== Description ==

CST One Click Disable Comments and RSS Feed **allows you to globally disable comments** on pages, posts, attachments and any post type in one click. It’s simple and powerful.

**It also removes all comment-related sections from admin panel and website frontend**.

If you find any issues, errors, or if you want to suggest new features to implement in later versions, use the appropriate
support form before issuing a negative comment so that we can resolve it.

We've included support links within the plugin settings page to make bug reporting easier


DETAILS

CST One Click Disable Comments and RSS Feed provide one option to completely disable the commenting feature in WordPress.
When you enable this option:

* All comments box from frontend website are hidden (in their place wordpress inserts the saying "Comments are closed");
* All comment related widgets are disabled;
* All comments links in Admin Menu are hidden;
* All comments links in Admin Bar are hidden;
* All comments related metabox “Recent Comments”, “Discussion” etc. are hidden from the WordPress Dashboard;
* Comments settings page will be hidden;
* All existing comments will be hidden
* All comments and trackbacks for any post type are hidden
* Any user trying to access comments page is redirected to the admin dashboard

CST One Click Disable Comments and RSS Feed provide an additional option to hide the saying "Comments are closed" from website frontend.

CST One Click Disable Comments and RSS Feed provide another additional option to completely disable the RSS Feed from your website
When you enable this option:

* All RSS/Atom feeds are disabled (and requests for these will show an error message);
* All pingbacks are disabled;
* The Pingback HTTP header is removed from header section;
* The RSS Feed HTTP header is removed from header section;

CST One Click Disable Comments disable all comments in your website, but in some cases there are issues that can overcome
this limitation, in which case just disable CST One Click Disable Comments plugin (because it hide comment menu ^_^), delete all existing comments and re-enable this plugin.

*As the name says, this plugin disables any comment system in one click, so it is not indicated if you need selective comment management (native wordpress feature)*


== Installation ==

1. Upload the plugin folder to the /wp-content/plugins/ directory or install it from wordpress repository
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The plugin settings can be accessed via the ‘Settings’ menu in the administration area
